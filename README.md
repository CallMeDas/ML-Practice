# ML Practice

This repository contains beginner-friendly machine learning practice notebooks built with Python and Jupyter.

## Topics Covered
- Linear Regression
- Logistic Regression
- Decision Tree
- Random Forest
- Naive Bayes
- SVM
- K-Means
- K-Fold Cross Validation
- One-Hot Encoding

## Quick Start
```powershell
.\env\Scripts\Activate.ps1
jupyter notebook
```

Open any `.ipynb` file and run the cells step by step.
